Restaurant Management System (Java Swing)
 Project Overview

This Restaurant Management System is a desktop-based application developed using Java Swing in Apache NetBeans. The system is designed to manage restaurant operations efficiently by providing role-based access for Manager and Employee.

The application helps in handling employee details, food menu management, order processing, and bill generation.

  User Roles
  Manager Login

The Manager has full access to the system:

Add, update, delete Employee Details

Add, update, delete Food/Menu Details

View all orders

Manage restaurant data

  Employee Login

The Employee has limited access:

Login using employee credentials

Take customer orders

Generate and print bills

View food menu (read-only)

  Technologies Used

Programming Language: Java

GUI Framework: Java Swing

IDE: Apache NetBeans

Database: MySQL 

JDBC: For database connectivity
